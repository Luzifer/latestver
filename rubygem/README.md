# Latestver RubyGem

Latestver command-line interface. Works with the hosted edition ([lv.binarybabel.org](https://lv.binarybabel.org)),
or private docker deployments. Project sync coming soon.

## Installation

    $ gem install latestver

## Usage

    $ latestver --help
